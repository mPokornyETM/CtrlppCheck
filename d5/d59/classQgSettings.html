public QgSettings::QgSettings ( const string &  id)

Generates a QgSettings class using the specified id.

The information required for this is read from the Settings Json Files under the path ".../data/qualityGates/settings/". Example id: "FunctionData.function.CCN"

Parameters
idSpecifies for which check the information should be read out
