Detailed Description

Lizard tool.

Lizard is Cyclomatic Complexity Analyzer. Lizard is used to calculate ctrl scripts and get the NLOC, Functions list, Function argument list, CCN and count of lines in function. See also https://github.com/terryyin/lizard
