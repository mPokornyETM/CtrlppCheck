How to deal with operating system dependencies in test cases?

You can use conditional statements based on the _ANDROID, _IOS, _OSX, _SOLARIS, _UNIX and _WIN32 symbolic constants, the value of which is either true or false, depending on the operating system being used.

Example

const string LINE_BREAK = _WIN32 ? "\r\n" : "\n";
