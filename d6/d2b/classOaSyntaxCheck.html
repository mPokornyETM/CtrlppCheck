Warning

Attention
It works only on platfaorm which has installed WinCC OA UI. For Linux platform is started with option '-platform offscreen' so it shall be possible to start it also without existing display.
Project must be registered an have a existing config file. Otherwise the the WinCC OA UI does not start.
