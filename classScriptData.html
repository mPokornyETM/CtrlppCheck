Detailed Description

Checks for static script data.

Script data are basic information about the script.

  • CCN
  • NLOC
  • Functions list
  • CountOfLines
  • Average values
Note
Call function calculate() before you want acces some file information. C-tor does not read the file to eliminate performacne.
