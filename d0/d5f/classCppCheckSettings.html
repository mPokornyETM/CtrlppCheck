◆ inconclusive

public bool CppCheckSettings::inconclusive = TRUE

option –inconclusive Allow that Cppcheck reports even though the analysis is inconclusive.

There are false positives with this option. Each result must be carefully investigated before you know if it is good or bad.

◆ inlineSuppressions

public bool CppCheckSettings::inlineSuppressions = FALSE

Enable in-line suppressions.

Use them by placing one or more comments, like: '// ctrlppcheck-suppress warningId' on the lines before the warning to suppress.
